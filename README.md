## 关于知食&#x20;

知“识”应该是知“食”，从“识”到“食”即是一个看到到咀嚼和品尝的过程。

借以知食的名义，梳理我对各种知识点的理解。

## 关于积累

为什么会有这个站？&#x20;

涉猎的东西越来越多后，发现过去的记忆变得模糊。

借用《三体》里面有一句我非常喜欢的话：

> 给岁月以文明，而不是给文明以岁月

这里在岁月中一点一滴地篆刻知识的文明吧。
