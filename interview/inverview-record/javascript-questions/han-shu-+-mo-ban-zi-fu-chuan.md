# 函数+模板字符串

输出是什么？

```javascript
function getPersonInfo(one, two, three) {
  console.log(one)
  console.log(two)
  console.log(three)
}

const person = 'Lydia'
const age = 21

getPersonInfo`${person} is ${age} years old`
```

{% tabs %}
{% tab title="选项" %}
* A: `"Lydia"` `21` `["", " is ", " years old"]`
* B: `["", " is ", " years old"]` `"Lydia"` `21`
* C: `"Lydia"` `["", " is ", " years old"]` `21`
{% endtab %}

{% tab title="答案" %}
**答案: B**

如果使用标记模板字面量，第一个参数的值总是包含字符串的数组。其余的参数获取的是传递的表达式的值！
